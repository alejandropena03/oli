## Bridge Protocol

Read `.bridge/README.md` for the full protocol. Summary:

- If `status: WAITING_FOR_LOCAL` → your turn. Pull, execute, validate, update status to `WAITING_FOR_CLAUDE`, push.
- If `status: WAITING_FOR_CLAUDE` → Claude Code's turn. Do not touch the active task.
- If `status: DONE` → archive task to `.bridge/tasks/TASK-NNN.md`, create next task.

When you finish a task:
1. Update `CURRENT_TASK.md` → `status: WAITING_FOR_CLAUDE`
2. Add entry to `HANDOFF_LOG.md`
3. `git add -A && git commit && git push`
